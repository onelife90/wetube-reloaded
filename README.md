# Wetube Reloaded

## Global Router

| 구분    | 내용   |
| ------- | ------ |
| /       | Home   |
| /join   | Join   |
| /login  | Login  |
| /search | Search |

## User Router

| 구분          | 내용              |
| ------------- | ----------------- |
| /users/:id    | See user          |
| /users/logout | Log out           |
| /users/edit   | Edit My Profile   |
| /users/remove | Remove My Profile |

## Video Router

| 구분               | 내용         |
| ------------------ | ------------ |
| /videos/:id        | See Video    |
| /videos/:id/edit   | Edit Video   |
| /videos/:id/delete | Delete Video |
| /videos/upload     | Upload Video |
